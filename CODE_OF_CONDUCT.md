# Code of Conduct

Be decent. Concretely:

- Critique code and ideas, never people. No harassment, slurs, or personal attacks — anywhere in issues, PRs, or commits.
- Assume good faith; disagree with specifics ("this breaks X because Y"), not vibes.
- No spam, no drive-by AI-generated issue/PR floods.
- Maintainers may edit, close, or remove anything that crosses these lines, and ban repeat offenders.

Report problems via a GitHub issue, or privately through [security advisories](https://github.com/cleanmcp/tess/security/advisories/new) if it involves a person.
